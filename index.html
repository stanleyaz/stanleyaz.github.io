Projects

Inviting you to check out some of my recent projects.

Churn Prediction for an Online Bank
In this project for Brass, an online banking startup, I trained a classifier to predict whether a customer is going to churn or not, potentially saving millions for the bank. This problem came with two caveats. First, the bank did not have an official definition of churn, and so I was responsible for exploring the data to define churn for my modeling. Second, due to security concerns, the bank did not share much demographic data, and so I had to engineer 11 new features for my modeling.

Predicting Poverty w/Satelitte Images
I (alongside two colleagues) predicted whether a given village in Africa suffers from extreme poverty by using only the geographic coordinates of that village and satellite imagery that can be freely and automatically downloaded using those coordinates. We trained convolutional neural networks and used ensemble modeling techniques to develop a production model that correctly identified extreme poverty in 79% of cases in which it appears while maintaining an overall predictive accuracy of 85% across all cases.
CHECK IT OUT
Rivian Automotive’s Subreddit Problem
In this fictional exercise for electric carmaker, Rivian, I collected 9000 rows of data from two subreddits - `r/cars` and `r/electricvehicles` - using the Pushshift API. Then I applied Natural Language Processing (NLP) to train a classifier on which subreddit a given post came from. My final stacked classifier model achieved accuracy and F1 scores of about 90%.
